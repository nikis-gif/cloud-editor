Learn Cloud
A simple flow for connecting and coding inside your game development workspace.
Cloud keeps your scripting flow clean. The plugin creates a private session, the app reads the workspace tree, and every save is sent back through the same controlled connection.
01Install CloudUse the desktop app for the smoothest experience, or open the web version when you need a quick fallback.
02Open the pluginInside your controlled game editor, open the Cloud plugin and create a fresh session.
03Copy credentialsCopy the Session ID and Hidden Secret. They are required together so the workspace stays private.
04Enter CloudPaste both values in the start panel. After validation, the Explorer loads your scripts and folders.
05Edit scriptsOpen scripts, use tabs, search the project, split the editor, and keep your flow focused.
06Sync changesPress Ctrl + S to save. Use Shift + Click for multi-selection, Delete to remove, and Ctrl + C / X / V for script operations.
