How Cloud works

A short setup guide for connecting your workspace and using the editor without friction.

Connect

  1. Open the plugin inside your game development environment.
  2. Create a private session.
  3. Copy the Session ID and Hidden Secret.
  4. Paste both values on the start screen and enter Cloud.

What happens behind the scenes

Cloud keeps a private session between the app and your development environment, which lets you browse, edit, save, and organize workspace files inside one focused flow.

  1. Open scripts, use tabs, split the editor, search the project, and keep your structure organized.
  2. Use Ctrl + S to save and the multi-select, copy, cut, paste, and delete actions to manage scripts faster.

Shortcuts

Ctrl + SSave the active script.
Shift + ClickSelect multiple scripts in the explorer.
Ctrl + C / X / VCopy, cut, and paste selected scripts or folders.
DeleteRemove selected scripts or folders.

A practical daily routine

  1. Connect once at the start of the session.
  2. Open the scripts you are actively working on.
  3. Save often and let Cloud keep the workspace current.
  4. Use Learn again whenever you want the setup steps nearby.

What you can manage in the workspace

  1. Create new scripts and folders.
  2. Rename, duplicate, move, and delete items.
  3. Copy, cut, and paste scripts across the workspace.
  4. Switch between tabs or open a split view when you need parallel editing.
