Learn Cloud
A clearer walkthrough for installing Cloud, connecting a session, and keeping your coding flow smooth.
What Cloud is doing Cloud keeps a private session between the app and your development environment, so your files stay inside one focused workflow.
Before you connect
  • Install the desktop app if you want the smoothest experience.
  • Open the plugin inside your game development environment.
  • Create a session and keep both credentials nearby.
Helpful shortcuts
Ctrl + SSave the active script.
Shift + ClickSelect multiple scripts.
DeleteRemove selected scripts or folders.
Ctrl + C / X / VCopy, cut, and paste scripts.
A simple path from setup to script editing. If you are opening Cloud for the first time, follow these steps once. After that, most of the work becomes open, edit, save, and keep moving.
01Install CloudUse the desktop app for the cleanest flow, or the web version when you need quick access.
02Open the pluginLaunch the plugin in your development setup and generate a fresh private session.
03Copy the credentialsTake the Session ID and Hidden Secret together. You will need both before the workspace can load.
04Enter CloudPaste the two values on the start screen and confirm the connection.
05Build inside the workspaceOpen scripts, use tabs, split the editor, search the project, and keep your structure organized.
06Save and manage filesUse Ctrl + S to save and the multi-select, copy, cut, paste, and delete actions to manage scripts faster.
A normal everyday routine
  • Connect once at the start of the session.
  • Open the scripts you are actively working on.
  • Save often and let Cloud keep the workspace current.
  • Use Learn again whenever you want the setup steps nearby.
What you can manage inside Cloud
  • Create new scripts and folders.
  • Rename, duplicate, move, and delete items.
  • Copy, cut, and paste scripts across the workspace.
  • Switch between tabs or open a split view when you need parallel editing.
